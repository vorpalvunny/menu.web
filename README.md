# VB-MENU.WEB

This is a project for VB-MENU.WEB.

## Download and Install

run script below:

```
git clone https://github.com/fizzjs/vb-menu.web.git
cd vb-menu.web
yarn install
```

## Development

Run development page on **localhost:8080**

```
$ npm run dev
```

## Build

Build for production.

```
$ npm run build
```
